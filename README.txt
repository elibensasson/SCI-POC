Dependencies of PoCPCP:
boost (apt install libboost-all-dev)
gtest (apt install libgtest-dev)
openssl (apt install libssl-dev)

The PoCPCP/scripts folder contains examples that show how to run the exhaustive search subset sum program, and the memory subset sum program.

The windows Code.sln was test with Visual Studio 2013, for convenience the dependencies can be downloaded from:
https://drive.google.com/file/d/0B0KFXc-NLCUcTnVJRXRxZW1DNVE/view?usp=sharing
This file should be extracted into the PoCPCP folder.
